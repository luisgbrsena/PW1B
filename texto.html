Textos e formatação

Abaixo você irá ver as tags de texto e formatação e suas funções:


  1. A Tag "ol" cria uma lista numerada.
  2. A Tag "ul" cria uma lista comum (não numerada).
  3. A Tag "p" cria um parágrafo.

    Este exemplo está sendo feito usando a tag "p".

  4. A Tag "br" (autofechante) quebra a linha.
    Ex: A Tag quebrará essa linha após a vírgula,
    e continuará na linha abaixo.
  5. A Tag "h1" serve para escrever títulos, a "h2" é para subtítulos, e podemos adicionar quantos h quisermos.
